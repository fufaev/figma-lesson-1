# ПРАКТИЧЕСКАЯ РАБОТА № 1

## «Основы работы в Figma для новичков»

**Цель:** освоить интерфейс Figma, сделать макет простого лендинга (сайт "Танцевальной студии") в Figma

**Задача:** 
* [Просмотреть видео](https://youtu.be/WU4JuG35XAQ);
* [Сделать форк репозитория с файлами по ссылке](https://github.com/fufaev/figma-lesson-1), реализовать лендинг простого сайта (танцевальной студии) используя знания, полученные в видеоуроке.
* Для продвинутых разработчиков на основе любого простого Figma Teplate реализовать вариант лендинга (тренажерный зал) со своими фотографиями, текстами на русском языке изменить телефоны на белорусские + карта расположения должна отбражать локацию в г. Минске.
